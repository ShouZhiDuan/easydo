《互联网平台企业涉税信息报送规定》

政策解读报告 - 营造公平环境，促进健康发展

📋政策出台背景与核心目标

为什么需要这项新规？

背景分析：

  • 平台经济发展迅速，在优化资源配置、推动产业升级等方面发挥积极作用
  • 传统税收监管方式难以适应平台经济的强流动性和高虚拟化特征
  • 线上线下税收监管存在信息不对称，影响税收公平

核心目标

  • 促进公平： 营造线上线下公平统一的税收环境
  • 提升效能： 提升税收服务与管理的效率和透明度
  • 保障权益： 保护合规纳税人的合法权益
  • 健康发展： 促进平台经济长期规范健康发展
💡 核心理念：这不是新税种，也不是加税，而是为了实现"信息透明，税收公平"。

🏢适用范围：谁需要报送？

互联网平台企业定义

为网络交易活动提供网络经营场所、交易撮合、信息发布等营利性服务的法人或非法人组织。

具体涵盖范围

一、报送主体范围

1. 境内平台

  • 所有具备交易闭环的电商平台（如淘宝、京东、抖音、美团、小红书等）
  • 取得ICP备案或许可证的平台企业

2. 境外平台

  • 境内有运营实体且获增值电信业务许可证的：由该实体报送
  • 境内有服务团队但未获证的：由负责招商、运营的境内供应商报送
  • 完全未设境内机构的：须委托境内代理人报送

3. 延伸主体

MCN机构、物流服务商、结算服务商等为平台内经营者提供盈利性服务的组织。

豁免情形：

  • 家政、外卖、网约车司机等便民劳务人员，若年收入≤12万元（未达个税起征点），免报收入信息
  • 政府共享信息已覆盖的数据无需重复报送

📊报送内容：需要报送什么信息？

两大核心信息类别

1. 身份信息

  • 已取得登记证照的经营者：
    • 名称、统一社会信用代码、地址
    • 店铺名称、唯一标识码、联系方式
  • 个人经营者/从业人员：
    • 姓名、证件类型、证件号码
    • 地址、店铺名称、联系方式

2. 收入信息

平台内经营者和从业人员上一季度在平台获取的与经营相关的收入。

🎯 目标：让税务机关掌握"谁在平台赚钱，赚了多少钱"的基础信息。

⏰报送时间：什么时候报送？

报送周期与时限

1. 平台基本信息

  • 首次报送：2025年7月1日-30日（所有现存平台必须完成）
  • 新设立平台：开展业务30日内报送
  • 信息变更：30日内更新

2. 经营者及收入信息

  • 首次报送：2025年10月1日-31日（报送2025年Q3数据）
  • 常规报送：每季度次月内完成（即每年1月、4月、7月、10月）

3. 历史数据

2025年6月20日前的信息无需追溯

🎁监管与处罚机制

1. 税务预警"双比对"系统

平台报送数据 vs 商家申报数据，触发条件包括：

  • 单季平台收入≥10万但申报为0
  • 年销售额≥500万未登记一般纳税人
  • 成本发票缺口＞30%

2. 处罚标准

违规情形 处罚措施
未按期报送 2万-10万元罚款
信息不实（瞒报/谎报） 5万-30万元罚款
情节严重（如隐匿收入） 10万-50万元罚款 + 停业整顿

3. 稽查案例（2025年7月已落地）

深圳税务局通过抖音数据比对，追查商家2023年隐匿收入。

📈影响分析：对商家的核心影响与应对建议

1. 影响范围

  • 超90%小微商家不受影响：月销售额≤10万免征增值税，年收入≤12万基本免个税
  • 重点监管对象：高收入低申报群体（如主播、跨平台经营者）、刷单虚构交易商家

2. 合规策略

  • 收入一致性：确保税务申报数据与平台流水匹配，差异率需＜10%
  • 凭证管理：保留交易记录、合同、物流单据至少5年
  • 主体优化：
    • 个体户：年所得≤200万可享个税减半
    • 有限公司：利润≤300万按5%缴企业所得税

🎯 核心要点总结：政策核心目标与趋势

  • 监管升级：从"被动申报"转向"平台数据主动穿透"，终结隐匿收入乱象
  • 公平税负：仅对违规高收入者强化征管，小微商户税负不变
  • 企业行动：7月30日前完成平台基本信息报送，10月起收入数据全面透明化

📅政策实施时间线

时间节点 重要事项
2025年6月20日 《规定》正式公布并施行
施行后30日内 平台企业需向税务机关报送基本信息
2025年第三季度起 开始按季度报送涉税信息
持续进行 每季度终了次月内完成报送

❓常见问题解答

Q1: 这是新增税种吗？
A: 不是。这只是信息报送要求，不涉及新税种或加税。目的是让税务机关更好地掌握平台经济的真实情况，确保税收公平。只有少数隐匿收入的高收入者会受到影响。
Q2: 外卖小哥、快递员需要担心吗？
A: 完全不需要担心。从事配送、运输、家政等便民劳务的人员，其收入信息免予报送，不会影响他们的工作和收入。
Q3: 平台企业需要做什么准备？
A: 需要建立涉税信息收集、核验和报送机制，确保按时向税务机关报送准确信息。
Q4: 如何保护个人信息安全？
A: 政策明确规定，税务机关应当对获取的涉税信息依法保密，建立涉税信息安全管理制度，落实数据安全保护责任。

📝总结

《互联网平台企业涉税信息报送规定》的出台，标志着我国在平台经济税收治理方面迈出了重要一步。这项政策以"信息透明、税收公平"为核心理念，在保护广大合规经营者和从业人员利益的同时，有效规范市场秩序，为平台经济的健康发展提供了有力保障。

特别值得注意的是，政策充分考虑了不同群体的实际情况，对便民劳务从业人员给予了明确的保护，体现了政策的人性化和精准化。
